01. 변수 : 데이터 불러오기

변수안에 저장된 데이터를 불러오는 방법입니다..

{
    let x = 100, y = 200, z = "javascript";     // 변수 한줄에 값을 한번에 지정

    console.log(x, y, z);
}
결과 확인하기
100, 200, javascript

반복문 : 변수를 이용해 반복적으로 값을 낼수있다.

for문에 사용되는 변수가 적은 것이 더 효율적이고 간단하므로 변수들의 관계를 잘 파악하여 불필요한 변수의 사용을 줄이는 것이 좋다.
